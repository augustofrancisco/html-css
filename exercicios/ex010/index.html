Criando um Link Interno

Os links internos vão fazer a relação à outras páginas do próprio site que está sendo acessado.
Exemplo: Você pode acessar a segunda página desse site clicando aqui.
Os links internos não precisam de uma URL para fazer o direcionamento, com tanto que a página destino esteja na mesma pasta do arquivo index.

Você também pode acessar a nossa página de notícias clicando aqui

Para direcionar o usuário a uma página dentro do site, mas que está localizada em uma subpasta, usa-se a tag <a>, no entanto, no atributo "href" devemos inserir o nome da subpasta, como no exemplo a seguir:

ex: <a href="noticias/pag003.html" rel="next" target="_self">exemplo de link </a>

Podemos também utilizar o atalho "ctrl + espaço" e o VSCode mostrará automaticamente as páginas e subpastas;

Para voltar de uma subpasta, no entanto, devemos utilizar o código "../" como no exemplo a seguir:

ex: <a href="a href="../index.html" rel="prev" target="_self">exemplo de link </a>

Target (Direcionamento)

  • O comportamento padrão de um hyperlink é abrir a página de destino na mesma aba em que você clicou no hyperlink;
  • No entanto, esse padrão não é adequado quando queremos que o usuário continue acessando a nossa página;
  • Por isso, foi criado o atributo target, que define onde será aberto a página do hyperlink clicado;

_blank

vai abrir o link em uma nova janela em branco, utilizado em links externos.

Exemplo: Acesse meu Instagram

_self

vai abrir o link na página utilizada no momento(padrão), utilizado para links internos.

Exemplo: Acesse meu Instagram

Atributo rel

O atributo rel foi criado para indicar a natureza do destino;

  • next -> indica que o link direciona para a próxima página do documento atual;
  • prev -> indica que o link direciona para uma página anterior do documento atual;
  • author -> indica que o link direciona para uma página do autor do artigo atual;
  • external -> indica que o link direciona para uma página que não faz parte do site atual
  • nofollow -> indica que é um site não endossado, como um link pago.

Links para Download

  • Download fotos em arquivo ZIP

Para fazer o link para download de um arquivo criamos uma âncora com a tagb <a> e no parâmetro href, apontamos para o arquivo desejado;

Basta que esse arquivo esteja na mesma pasta do arquivo index ou em uma subpasta;

Exemplo:

<a> href="fotos/formatura.rar" download="formatura.rar" type="application/zip">Download fotos em arquivo ZIP</a>

Parâmetros:

  • download -> indica o arquivo que será baixado;
  • type -> indica o tipo do arquivo que será baixado.
