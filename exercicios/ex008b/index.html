Fonte Monoespaçada

A fonte monoespaçada é usada, geralmente, para que a leitura de um código fonte, por exemplo, possa ser facilitada.

Ex:

document.getElementbyID('teste') é escrito na linguagem JavaScript. (escrita normal)

document.getElementbyID('teste') é escrito na linguagem JavaScript. (monoespaçada)

Para isso, utilizamos a tag code.

No entanto, se digitarmos um código em qualquer linguagem, devemos utilizar a tag pre, pois caso contrário, o código não ficará disposto da maneira em que foi digitado no editor.

Sem a tag pre:

def verificar_par(numero): if numero % 2 == 0: return True else: return False numero = int(input("Digite um número inteiro: ")) if verificar_par(numero): print(numero, "é um número par.") else: print(numero, "não é um número par.")

Com a tag pre:

        
def verificar_par(numero):
    if numero % 2 == 0:
        return True
    else:
        return False
numero = int(input("Digite um número inteiro: "))
if verificar_par(numero):
    print(numero, "é um número par.")
else:
    print(numero, "não é um número par.")
