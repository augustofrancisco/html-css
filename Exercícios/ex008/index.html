Marca Texto

Podemos criar também, um texto marcado utilizando a tag MARK

Para mudar a cor da marcação, devemos criar um estilo para a tag mark no cabeçalho. Dessa forma, todas as vezes que usarmos a tag mark, aparecerá da cor que definimos no cabeçalho.
